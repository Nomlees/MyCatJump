# MyCatJump
Цель:
- Преодоление препятствий
- Сбор лакомства
- Улучшение результатов

Реализовано:
- Меню игры
- Кот, который сам пока не бежит
- Прыжок кота, который не имеет смысла
- Лакомство

Планируется:

- Анимировать кота
- Добавить счёт
